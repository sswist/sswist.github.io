Welcome to the Python Basics Game

This is a Python game that teaches players the basics of Python programming language.

If you need to install Python on your computer, you can download it here.

To download and play the game, click the button below:

Play the Game

Open terminal or command prompt on your computer and enter the following command: 

        
        python PGame.py

If that does not work you may need to enter: 

        
        python3 PGame.py
